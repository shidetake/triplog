# カテゴリルール

## 使用可能な語彙（これ以外使わない）

- `渡航`
- `現地移動`
- `飲食`
- `衣服`
- `エンタメ`
- `お土産`
- `宿泊`
- `通信`
- `医療`
- `カジノ`
- `その他`

11種類のみ。新しいカテゴリは勝手に追加しない。`日用品` は廃止済み（スーパー=`飲食`、ドラッグストア=`医療`、雑貨店=`その他` 等、一番近いものへ寄せる）。

---

## マッピング指針

| 例 | カテゴリ |
|---|---|
| 航空会社（Delta, ANA, JAL等） | `渡航` |
| 成田エクスプレス・空港バス・空港リムジン | `渡航` |
| ホテル直接請求（客室・Resort Fee・宿泊税） | `宿泊` |
| ホテル内レストランの **部屋付け** | `飲食`（宿泊ではない） |
| Uber / Lyft / タクシー / 路線バス | `現地移動` |
| レストラン・ブルワリー・バー・カフェ | `飲食` |
| ABC Stores等のコンビニ | 中身次第で `飲食`（食料品）or `お土産` |
| スーパーマーケット（Whole Foods, Safeway, Times Super, Waikiki Market 等） | `飲食`（食料品中心） |
| ドラッグストア（Longs, CVS, Walgreens 等） | `医療`（薬・トイレタリ中心） |
| ディスカウント／雑貨店（Don Quijote, Target, Walmart 等） | `その他` |
| デパート・ブランドショップ | `衣服` or `お土産` |
| ツアー（Star of Honolulu等）・入場料・水族館 | `エンタメ` |
| 病院・クリニック・薬局調剤 | `医療` |
| カジノ・スロット・ポーカールーム | `カジノ` |
| Wi-Fiルーター・ローミング | `通信` |
| 上記いずれにも当てはまらない | `その他` |

---

## マーチャント名 → カテゴリのヒント辞書

`scripts/src/categorize.ts` の `MERCHANT_CATEGORY_MAP` で管理。
未登録のマーチャントは LLM で判定するか、ユーザーに確認する。
**勝手に「飲食」等にデフォルトしない。**
